Для запуска проекта нкжно зайти в папку через консоль cmd Converter и активировать виртуальное окружение venv командой .\venv\Scripts\activate
После перейти в папку app опять-же через консоль и выполнить команду python .\manage.py runserver 
Затем перейти по адресу http://127.0.0.1:8000/ запущенного сервера
